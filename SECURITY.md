# Security policy

## Supported versions

Security fixes are released for the latest published version of Batchwork.

| Version          | Supported |
| ---------------- | --------- |
| 0.1.x            | Yes       |
| Earlier versions | No        |

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability.

Report it privately through [GitHub private vulnerability reporting](https://github.com/ajanraj/batchwork-ai/security/advisories/new) or email [hey@ajanraj.com](mailto:hey@ajanraj.com).

Include the affected version, impact, reproduction steps, and any suggested mitigation. Do not include live provider credentials, webhook secrets, customer data, or other sensitive material.

We aim to acknowledge reports within three business days and will share remediation progress while the issue remains under embargo. Please allow time for a fix and release before public disclosure.
